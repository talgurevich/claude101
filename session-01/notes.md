# מפגש 01 — יסודות, Skills, והנחיה

## עיקרי המפגש

1. **Setup** — התקנה, סשנים, הרשאות, CLAUDE.md, פקודות slash
2. **Skills** — מודולי מומחיות לשימוש חוזר. שלוש דוגמאות חיות: code review, security audit, UI
3. **הנחיה** — מסגרת בת 10 רכיבים מבוססת על דוגמה ויזואלית של Anthropic. תרגול נאיבי מול מובנה
4. **סיכום ושאלות** — takeaways, מה במפגש הבא, Q&A

## Takeaways

- Claude Code הוא לא עוד צ'אט. זו לולאת סוכן שיכולה לקרוא, לערוך, להריץ ולבדוק קוד אוטונומית.
- הקובץ `CLAUDE.md` הוא הזיכרון של הסוכן בפרויקט שלך. בלעדיו - כל סשן מתחיל מאפס.
- Skills הופכים את הסוכן ממהיר ליעיל. מודולי מומחיות לשימוש חוזר.
- ההבדל בין פרומפט נאיבי לפרומפט מובנה הוא לא 10% איכות. לעיתים זה ההבדל בין תשובה לא שימושית לבין תשובה מצוינת.
